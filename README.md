# Django Store Project
Учебный проект интернет-магазина на Django.

### Стек:
- Python 3.x, Django 4.2+
- PostgreSQL / SQLite
- Bootstrap 5, AJAX/jQuery
- Регистрация, корзина, оформление заказов.

### Запуск:
1. `pip install -r requirements.txt`
2. `python manage.py migrate`
3. `python manage.py runserver`
